## Installation & Setup Guide for TestDisk & PhotoRec

#### System Compatibility

- **Supported Windows versions:** Windows 7, 8, 10, 11 (32-bit and 64-bit)
- **Architecture support:** Compatible with both x86 and x64 systems
- **Additional Requirements:** Administrator privileges for disk-level operations
- **Disk space required:** Approximately 10-15 MB for the application

#### Installation Methods

**Direct Installation:**

1. Download the TestDisk and PhotoRec package for Windows.
2. Extract the downloaded ZIP archive to a folder of your choice.
3. No installation required; run `testdisk_win.exe` or `photorec_win.exe` directly.
4. Launch from the extracted folder using administrator privileges.

**Portable Usage:**

1. Copy the extracted folder to a USB drive for portable operation.
2. Run the executables from any Windows computer without installation.

#### First-Time Setup & Configuration

1. **Launch TestDisk:** Run `testdisk_win.exe` with administrator privileges.
2. **Create Log:** Choose whether to create a log file for recording recovery operations.
3. **Select Disk:** Choose the disk containing lost or damaged partitions from the list.
4. **Partition Table Type:** Select the correct partition table type (Intel/PC for most Windows systems).

#### Core TestDisk Operations

1. **Analyze Partition Structure:**
   - TestDisk scans the selected disk for existing and lost partitions.
   - Review detected partitions and their status (deleted, present, etc.).
   - Use Quick Search or Deeper Search for more thorough scanning.

2. **Partition Recovery:**
   - Select partitions to recover from the scan results.
   - Write recovered partition table to disk after verification.
   - Reboot to apply changes and access recovered partitions.

3. **Boot Sector Repair:**
   - Rebuild damaged or corrupted boot sectors.
   - Fix MBR (Master Boot Record) or boot sector code.
   - Restore backup boot sector if available.

#### PhotoRec File Recovery Operations

1. **Launch PhotoRec:** Run `photorec_win.exe` with administrator privileges.
2. **Select Disk:** Choose the storage device from which to recover files.
3. **Select Partition:** Choose the specific partition or "Whole disk" for scanning.
4. **File Type Selection:** Select which file types to recover (documents, photos, videos, etc.).
5. **Destination Folder:** Choose where to save recovered files (different from source drive).

#### Key Features & Advanced Options

- **Partition Table Recovery:** Recover lost or deleted partitions from damaged or formatted drives.
- **MBR Repair:** Fix corrupted master boot records to restore system bootability.
- **File Carving:** PhotoRec recovers files based on file signatures, bypassing file system corruption.
- **Multi-Platform Support:** Available for Windows, macOS, Linux, and other operating systems.
- **Open Source:** Free to use with transparent, community-reviewed code.

#### Troubleshooting

- **Partition Not Found:** Run Deeper Search for more thorough scanning. Check that correct disk is selected.
- **Recovered Files Corrupt:** Some files may be partially overwritten. Try recovering to a different destination drive.
- **Boot Sector Repair Fails:** Verify partition alignment and disk geometry. Use backup boot sector if available.
- **Slow Recovery Speed:** Limit file types in PhotoRec to reduce scan time. Recover to a different physical drive.

#### Updates & Maintenance

- TestDisk and PhotoRec are actively maintained with regular updates.
- Manual updates require downloading the latest version from the official website.
- No installation needed; simply replace existing files with newer versions.

#### Support Resources

- Built-in documentation available via the application and website.
- Official website provides detailed guides, FAQs, and user forums.
- Community support for specific recovery scenarios and disk issues.

## About TestDisk & PhotoRec

TestDisk and PhotoRec are powerful open-source data recovery tools designed for recovering lost partitions and files from damaged storage devices. TestDisk specializes in partition recovery software operations, rebuilding corrupted partition tables, repairing boot sectors, and recovering lost partitions. The disk recovery utility works with multiple file systems including NTFS, FAT, exFAT, and ext2/3/4. PhotoRec complements this as a file recovery utility that recovers individual files based on their signatures, making it effective even when the file system is severely damaged. Together, these disk diagnostic utility tools provide a comprehensive solution for data restoration across hard drives, SSDs, memory cards, and USB devices. As reliable file system recovery tools, they serve as essential utilities for users needing to recover data from formatted drives, repartitioned disks, or corrupted storage media.
